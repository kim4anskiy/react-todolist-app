# React-приложение "Список задач" (Todo List)

## Описание проекта
Интерактивное веб-приложение для управления задачами, разработанное на React. 
Позволяет добавлять, удалять и отмечать выполненные задачи.

## Функционал
- Добавление новой задачи
- Отметка задачи как выполненной (перечеркивание)
- Удаление задачи

## Технологии
- React
- JavaScript (ES6+)
- CSS3
- HTML5
